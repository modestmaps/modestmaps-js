Usage

Using modestmaps.js is as simple as including either the full source (about 100K, useful for debugging) or the minified source (about 36K, for production environments) in the <head> of your HTML document:

<script type="text/javascript" src="modestmaps.js"></script>

MM.Map

The Map class is where it's at. Maps have layers, a center and zoom, dimensions, and event handlers that manage interaction.

First, you'll need to add a parent element for your map in your HTML:

<div id="map"></div>

Then, create your tile layer, define some dimensions, and make your map:

var layer = new MM.TemplatedLayer("http://tile.stamen.com/toner/{Z}/{X}/{Y}.png");
var size = new MM.Point(640, 480);
var map = new MM.Map("map", layer, size);

You can set the map's center and zoom:

map.setCenter(new MM.Location(37.764, -122.419));
map.setZoom(12);
MM.Map(parent [, layerOrLayers [, dimensions [, eventHandlers ]]])
ArgumentTypeDescription
parent HTMLElement|String

The parent element of the map. This can be either an object reference or the ID of the element (as a string, e.g. "map").

If an element with the provided ID string is not found, an exception is raised.

If parent is not an object or a string, an exception is raised.
layerOrLayers Object|Array

Either a single layer object or an array of layer objects. Layer objects should be Layer instances or implement the Layer interface.

Before version 3.1.1, an exception was raised if layerOrLayers was not an object or an array.
dimensions Point

An optional map size in pixels, expressed as an Point instance (or any object with x and y attributes).

If no dimensions are provided, the dimensions are derived from the width and height of the parent element, and the map's autoSize flag is set to true.

Empty <div> elements have no intrinsic height, so if you don't provide dimensions you'll need to provide the map's parent height in CSS (either inline or in a stylesheet).
eventHandlers Array

An optional array of interaction event handlers, which should implement the EventHandler interface. If no handlers are provided (eventHandlers === undefined), the map is initialized with mouse and touch handlers.

To initialize the map without any interaction handlers, use null or an empty array ([]).

Getter/Setter Methods

Getter/setter methods behave differently depending on whether they receive arguments: The "getter" form (with no arguments) returns the current value, and the "setter" form sets it to the value provided then returns this, which makes function chaining possible.

Properties
PropertyTypeDescription
autoSize Boolean

The autoSize property is set to true if no size is provided in the constructor. When autoSize is true, the map's dimensions will be recalculated on window resize.
callbackManager CallbackManager The map's callback manager.
coordinate Coordinate The map's current center coordinate.
coordLimits Array

An array specifying the map's coordinate bounds, in which the first element defines the top left (northwest) and outermost zoom level, and the second defines the bottom right (southwest) and innermost zoom.

You can adjust the minimum and maximum zoom levels of the map without affecting the bounds with setZoomRange().

You can constrain a coordinate to the map's limits with enforceLimits().
eventHandlers Array The map's interaction event handlers.
dimensions Point The map's current dimensions, expressed as a Point instance.
layers Array An array of the map's layers.
parent HTMLElement The map's parent (container) DOM element.
projection Projection The map's projection, or CRS.
tileSize Point The size of the map's image tiles, expressed as a Point instance. By default, tiles are 256 pixels square.
